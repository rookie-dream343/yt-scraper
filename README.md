# YouTube 视频下载爬虫

一个简单易用的YouTube视频下载工具，支持多种清晰度选择。

## 功能特点

- 支持选择视频清晰度 (1080p, 720p, 480p等)
- 自动获取视频信息
- 支持下载字幕
- 显示下载进度
- 自动合并音视频
- 友好的命令行交互界面

## 安装

### 方法1: 使用 pip 安装依赖

```bash
pip install -r requirements.txt
```

### 方法2: 直接安装 yt-dlp

```bash
pip install yt-dlp
```

### 方法3: 使用 ffmpeg (用于合并音视频)

下载高质量视频(1080p+)时需要ffmpeg来合并音视频：

**Windows:**
```bash
# 使用 winget
winget install ffmpeg

# 或从官网下载: https://ffmpeg.org/download.html
```

**Mac:**
```bash
brew install ffmpeg
```

**Linux:**
```bash
sudo apt install ffmpeg  # Ubuntu/Debian
sudo yum install ffmpeg  # CentOS/RHEL
```

## 使用方法

### 运行程序

```bash
python yt_downloader.py
```

### 操作流程

1. 输入保存路径 (直接回车使用默认 `./downloads`)
2. 输入YouTube视频链接
3. 查看视频信息和可用清晰度
4. 选择想要的清晰度
5. 选择是否下载字幕
6. 等待下载完成

### 清晰度选择

| 选项 | 说明 |
|------|------|
| `0` 或回车 | 最佳质量 (推荐) |
| `1-N` | 选择对应序号的清晰度 |
| `b` | 仅下载最佳单文件 |
| `q` | 退出程序 |

### 支持的链接格式

- `https://www.youtube.com/watch?v=xxxxx`
- `https://youtu.be/xxxxx`
- YouTube Shorts链接
- YouTube播放列表 (会下载所有视频)

## 注意事项

1. 下载受版权保护的内容请遵守当地法律法规
2. 下载速度取决于网络连接
3. 1080p以上清晰度需要安装ffmpeg
4. 部分视频可能受地区限制无法下载

## 常见问题

**Q: 下载的视频没有声音？**
A: 选择带有"✓"标记的格式，或安装ffmpeg后使用默认选项下载。

**Q: 下载失败？**
A: 检查网络连接，确认链接有效，或尝试更新yt-dlp版本。

**Q: 如何下载播放列表？**
A: 直接输入播放列表链接即可，程序会自动处理。
